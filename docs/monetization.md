## Black holes (paid feature)

Private, bounded canvases within the void. Same core experience — isolated, owned, customizable.

### How it works

1. User pays a one-time fee (crypto or other anonymous method)
2. A token is generated server-side and returned
3. Token is the key — owner shares it however they want (DM, note, verbally)
4. Recipient pastes token into the Black Holes portal to enter
5. Owner can also generate a separate view-only link for public sharing

### Pricing model

- Base price covers a chosen duration (owner sets at purchase)
- Larger size = higher price
- Renewal available before the hole collapses
